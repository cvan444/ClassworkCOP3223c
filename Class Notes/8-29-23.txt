Types of errors in C -----

Syntax error - forget to add.... ; or ) or }
run time error - /0... something you cant do  
logic errors - do not get expected results. No advice from compiler




programming styles ------

Include ample white space
Indent all code inbetween functions
Use a header comment to list items such as, author, date purpose, 
Place comments in every major section 
Make your vars meaningful 
use uppercase for const 
Be consistent 



Rules for identifiers (vars)------

must be comprised of letters, digits, and/or underscores
a letter or _ has to be the first digit,letter
in some instances casing matters
must not be a keyword
